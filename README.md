# 📚 Catalogando - Projeto de Estudos (Android/Kotlin)

Olá! 👋 Este é o "Catalogando", um aplicativo focado no gerenciamento de acervos (Livros, Vinis e CDs). 

Sou "estudante" (recém formado em Técnico em Desenvolvimento pela ETEC PHILADELPHO GOUVEA NETTO e um curioso e uso meu GitHub como um diário de bordo para registrar projetos pessoais. 
Construí este projeto do zero para aplicar, na prática, os conceitos de desenvolvimento Mobile nativo.

## 🎯 Objetivo do Projeto
Sair da teoria e entender como um aplicativo Android real funciona por baixo dos panos. O foco principal foi aprender a conectar uma interface moderna com um banco de dados local.

## 🧠 O que eu aprendi construindo este app:
Este projeto foi um grande laboratório de testes para mim. Durante o desenvolvimento, consegui colocar em prática:
* **Jetpack Compose:** Aprendi a construir telas responsivas e suporte a Dark Mode usando a nova forma de criar interfaces no Android.
* **Room Database (SQLite):** Entendi como criar tabelas, fazer consultas (CRUD) e salvar dados na memória interna do celular de forma segura.
* **Recursos Nativos:** Consegui integrar a câmera e a galeria de fotos nativa do dispositivo.
* **Arquitetura Básica:** Uso de Coroutines e Flow para manter o aplicativo rápido, mesmo fazendo buscas no banco de dados.
(Usando IA GEMINI)

## 🛠️ Stack Tecnológica
* Kotlin
* Jetpack Compose
* Room Database
* Material Design 3
